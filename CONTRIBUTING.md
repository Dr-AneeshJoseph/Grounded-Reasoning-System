GRS 11.0 prioritizes **Control** and **Transparency**. Any pull request that breaks the "Trace" or makes the system "hallucinate confidently" will be rejected.

### 1. The "Haiku Test" (Context Awareness)
*Objective: Ensure the system knows when to stay quiet.*
* **Input:** "Write a haiku about logic."
* **Success Condition:** The system output is **clean**. It should NOT display the full granular `MT:` trace for such a simple (C1) task, or it should default to the `[Narrative_Weaver]` without a lecture.
* **Failure Condition:** The system triggers a "Deep Think" loop and writes a 5-paragraph analysis before the haiku.
